# Spam Email Detection using Machine Learning

## Project Overview
This project classifies SMS messages as Spam or Ham using Machine Learning algorithms.

## Algorithms Used
- Logistic Regression
- Random Forest Classifier

## Features
- Data preprocessing
- TF-IDF vectorization
- Train/Test split
- Cross-validation
- Accuracy comparison
- Precision, Recall, F1-Score
- ROC-AUC Evaluation
- Confusion Matrix
- ROC Curve

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Results
Random Forest achieved slightly better performance compared to Logistic Regression.
